## `createRepoLens(options?)`

Returns a record of AI SDK tools you can pass to `generateText` or `streamText`. All options are optional — the SDK reads `GITHUB_TOKEN` from your environment by default:

```ts [types.ts]
type RepoLensOptions = {
  token?: string
  requireApproval?: boolean | Partial<Record<GithubWriteToolName, boolean>>
  preset?: RepoLensPreset | RepoLensPreset[]
}

type RepoLensPreset =
  | 'code-review'
  | 'issue-triage'
  | 'repo-explorer'
  | 'maintainer'
```

Minimal usage — reads `GITHUB_TOKEN` automatically:

```ts [minimal.ts]
import { createRepoLens } from '@repolens-ai/sdk'

const tools = createRepoLens()
```

With a preset and explicit token:

```ts [with-options.ts]
import { createRepoLens } from '@repolens-ai/sdk'

const tools = createRepoLens({
  token: 'github_pat_xxxxxxxxxxxx',
  preset: 'repo-explorer',
})
```

See [Scope with presets](/guide/presets) for preset details and [Control write safety](/guide/approval-control) for approval options.

## `createGithubAgent(options)`

Returns a `ToolLoopAgent` with RepoLens and system instructions pre-configured. The token is also auto-detected from `GITHUB_TOKEN`:

```ts [types.ts]
type GithubAgentOptions = {
  model: string
  token?: string
  preset?: RepoLensPreset | RepoLensPreset[]
  requireApproval?: boolean | Partial<Record<GithubWriteToolName, boolean>>
  system?: string
}
```

Use this when you want:

- reusable `.generate()` / `.stream()` calls across multiple prompts
- preset-aware system instructions without manual wiring
- a centralized agent definition shared across your codebase

```ts [agent.ts]
import { createGithubAgent } from '@repolens-ai/sdk'

const agent = createGithubAgent({
  model: 'anthropic/claude-sonnet-4.6',
  preset: 'code-review',
  system: 'You review PRs for security issues. Cite file paths and line numbers.',
})
```

## `createOctokit(token?)`

Returns a configured [`@octokit/rest`](https://github.com/octokit/rest.js) instance. Use this when you need lower-level GitHub API access or want to build custom tool factories:

```ts [custom-tool.ts]
import { createOctokit } from '@repolens-ai/sdk'

const octokit = createOctokit()
const { data } = await octokit.repos.get({ owner: 'NeoPilot', repo: 'repolens-ai' })
```
