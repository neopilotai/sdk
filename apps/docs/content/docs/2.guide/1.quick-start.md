## Install everything at once

If you haven't completed [Installation](/getting-started/installation) yet:

:::code-group
```bash [pnpm]
pnpm add @repolens-ai/sdk ai zod
```
```bash [npm]
npm install @repolens-ai/sdk ai zod
```
```bash [yarn]
yarn add @repolens-ai/sdk ai zod
```
```bash [bun]
bun add @repolens-ai/sdk ai zod
```
:::

## Use tools directly

The simplest integration: pass all tools to `generateText` and let the model pick the right GitHub operations:

```ts [list-prs.ts]
import { createRepoLens } from '@repolens-ai/sdk'
import { generateText } from 'ai'

const { text } = await generateText({
  model: 'anthropic/claude-sonnet-4.6',
  tools: createRepoLens(),
  prompt: 'List open pull requests on vercel/ai and summarize each one.',
})

console.log(text)
```

The SDK reads `GITHUB_TOKEN` from your environment automatically. This gives the model access to all 18 tools â€” see the [Tools Catalog](/api/tools-catalog) for what each one does.

## Narrow tools with a preset

When you want to limit which GitHub operations are available, use a preset. This gives the model only pull request and commit tools:

```ts [review-only.ts]
import { createRepoLens } from '@repolens-ai/sdk'
import { streamText } from 'ai'

const result = streamText({
  model: 'anthropic/claude-sonnet-4.6',
  tools: createRepoLens({ preset: 'code-review' }),
  prompt: 'Review the latest PR on repolens-ai/sdk for security issues.',
})

for await (const chunk of result.textStream) {
  process.stdout.write(chunk)
}
```

Learn how each preset maps to tools in [Scope with Presets](/guide/presets).

## Wrap tools in a reusable agent

When you need the same configuration across multiple calls, use `createGithubAgent` to get a persistent agent:

```ts [triage-agent.ts]
import { createGithubAgent } from '@repolens-ai/sdk'

const triager = createGithubAgent({
  model: 'anthropic/claude-sonnet-4.6',
  preset: 'issue-triage',
  system: 'You classify issues as bug, feature, or question. Always post a comment with the classification.',
})
```

::tip
Start with `createRepoLens()` to learn the tools. Add a preset when you want focus. Move to `createGithubAgent()` when you need reusable behavior.
::
